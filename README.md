### Tecnologias usadas no frontend:

- React 
- Typescript 
- Bootstrap
- Deploy: Netlify

### Tecnologias usadas no backend:

- Spring Boot com BD relacional
- PostgreSQL
- Postman
- Deploy: Heroku

## Usando a aplicação
Para usá-la, basta executar os seguintes comandos na raíz do projeto:
```
npm install
npm start
```
